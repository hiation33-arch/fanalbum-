# FanAlbum 사용가이드
**made by 바람**

---

## 시작하기

`FanAlbum_배포용.html` 파일을 브라우저로 열면 바로 시작할 수 있어요.  
인터넷 없이도 사용 가능하며, 사진은 내 기기에만 저장돼요.

---

## 화면 구성

### 🏠 대문
앱을 처음 열면 커버 카드가 보여요.
- **커버 사진** — 우측 상단 📷 버튼으로 변경
- **앨범 제목** — 직접 클릭해서 편집
- **연도** — 연필 아이콘으로 편집
- **포토앨범 만들기** — 갤러리로 이동

---

## 핵심 기능

### 📷 사진 추가
- 헤더의 **+ 사진 추가** 버튼 클릭
- 또는 화면에 사진 파일을 **드래그 앤 드롭**
- ✨ 카테고리를 선택한 상태에서 추가하면 해당 카테고리로 자동 분류

### 🗂️ 카테고리 분류
1. 사이드바 **카테고리 +** 버튼으로 카테고리 추가
2. 사진 클릭 → **편집** → 카테고리 선택
3. 사이드바에서 카테고리 클릭 → 해당 사진만 표시

### 🔍 검색
상단 검색창에 **사진 이름 또는 메모** 입력 → 실시간 필터링

### ✏️ 사진 편집
사진 카드 → 편집 버튼 클릭
- 제목, 카테고리, 날짜, 메모 편집
- 필터 8종 적용 (흑백, 세피아, 비비드 등)
- 밝기 · 대비 · 채도 조정

### ☑️ 다중 선택
헤더 **다중 선택** 버튼 → 카테고리 필터로 원하는 사진만 표시 → 선택 → 카테고리 일괄 지정 또는 삭제

## 포토카드 편집

**포토카드 편집** 메뉴에서 인쇄용 포토카드를 꾸밀 수 있어요.

| 기능 | 설명 |
|------|------|
| 사진 추가 | 왼쪽 소스 패널에서 사진 클릭 또는 캔버스에 드래그 |
| 배경 사진 설정 | 소스 패널 썸네일 하단 **배경** 버튼 클릭 |
| 배경 사진 제거 | 소스 패널 상단 **🖼 배경 설정됨 [제거]** 버튼 클릭 |
| 이동 | 캔버스 위 사진 드래그 |
| 크기 조절 | 사진 우측 하단 핸들 드래그 |
| 회전 | 툴바 🔄 버튼 (15° 단위) |
| 앞으로 | 툴바 레이어 버튼 |
| 삭제 | 툴바 ✕ 버튼 |
| 텍스트 추가 | 툴바 **텍스트** 버튼 → 문구 입력 |
| 레이아웃 | 2×2 / 3×3 / 히어로 / 스트립 자동 정렬 |
| 배경색 | 툴바 색상 박스 클릭 |
| 페이지 추가 | 탭 옆 **+** 버튼 |

---

## 포토카드 출력

**포토카드 출력** 메뉴에서 완성된 포토카드를 저장하거나 출력할 수 있어요.

- **인쇄 사이즈** — A5 / A4 / 정사각 / 엽서 / 폴라로이드 선택
- **이미지 자르기** — 출력 전 원하는 영역만 선택
- **이미지 저장** — PNG 파일로 저장 (그림자 효과 포함)
- **포토카드 출력** — 프린터로 바로 출력

## 테마 변경

사이드바 하단 **테마** 버튼 → 6가지 무드 선택

| 테마 | 분위기 |
|------|--------|
| 🌊 바다 | 파스텔 블루 (기본) |
| 🌸 벚꽃 | 핑크 파스텔 |
| 🌿 포레스트 | 그린 민트 |
| 🌙 나이트 | 다크 모드 |
| ☀️ 골든 | 따뜻한 앰버 |
| 🖤 모노 | 흑백 미니멀 |

## 데이터 백업

사이드바 하단에서 백업 관리가 가능해요.

- **백업 내보내기** — JSON 파일로 저장
- **백업 가져오기** — 저장한 JSON 파일로 복원

> ⚠️ HTML 파일을 직접 열면(`file://`) 새로고침 시 데이터가 초기화될 수 있어요.  
> 영구 저장은 Netlify 등 웹 호스팅 서비스를 이용하거나, 정기적으로 백업하세요.
