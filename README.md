# FanAlbum ✦

나만의 팬 다이어리 — 사진을 모으고, 꾸미고, 포토카드로 출력까지.

> **made by 바람**

---

## 소개

FanAlbum은 순수 HTML 단일 파일로 동작하는 팬앨범 웹앱이에요.  
설치 없이 브라우저에서 바로 열 수 있고, 사진은 내 기기에만 저장돼요.

---

## 시작하기

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있어요.  
인터넷 연결 없이도 동작합니다.

---

## 주요 기능

### 📷 갤러리
- 사진 추가 (버튼 클릭 또는 드래그 앤 드롭)
- 카테고리 분류 및 필터링
- 제목·메모·날짜 편집
- 필터 8종 (흑백, 세피아, 비비드 등) + 밝기·대비·채도 조정
- 다중 선택 → 일괄 카테고리 지정 / 삭제
- 실시간 검색

### 🎨 포토카드 편집
- 사진 자유 배치, 크기 조절, 회전
- 텍스트 추가
- 레이아웃 자동 정렬 (2×2 / 3×3 / 히어로 / 스트립)
- 배경색·배경사진 설정
- 페이지 추가

### 🖨️ 포토카드 출력
- 인쇄 사이즈 선택 (A5 / A4 / 정사각 / 엽서 / 폴라로이드)
- PNG 이미지로 저장
- 프린터 직접 출력

### 🎨 테마
| 테마 | 분위기 |
|------|--------|
| 🌊 바다 | 파스텔 블루 (기본) |
| 🌸 벚꽃 | 핑크 파스텔 |
| 🌿 포레스트 | 그린 민트 |
| 🌙 나이트 | 다크 모드 |
| ☀️ 골든 | 따뜻한 앰버 |
| 🖤 모노 | 흑백 미니멀 |

### 💾 데이터 백업
- JSON 파일로 백업 내보내기 / 가져오기

---

## 주의사항

> HTML 파일을 직접 열면(`file://`) 새로고침 시 데이터가 초기화될 수 있어요.  
> 영구 저장은 Netlify 등 웹 호스팅 서비스를 이용하거나, 정기적으로 백업하세요.
